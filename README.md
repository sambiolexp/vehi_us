Proyecto de Sprint 7

# Análisis de Datos de Vehículos

## Descripción del Proyecto
En este Sprint construí y desplegué un panel de control de una app web en un servicio en la nube.
Esta aplicación web permite explorar un conjunto de datos de anuncios de venta de coches a través de visualizaciones interactivas.

## Funcionalidades
Se agregaron 2 diferentes gráficos para la visualización de los datos:
- Histograma del odómetro de los vehículos
- Gráfico de dispersión que muestra la relación entre el odómetro y el precio

## Tecnologías Utilizadas
- Python
- Streamlit
- Plotly Express
- Pandas
